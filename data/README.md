# Sample Data

This directory contains sample data files for testing and demonstration purposes.

## Available Samples

- **sample_annual_report.pdf** - Sample company annual report
- **gst_sample.csv** - Sample GST data
- **bank_statement.csv** - Sample bank statement data

## Usage

These files can be used to test the document processing pipeline and credit assessment features without requiring real company data.

## Note

All sample data is synthetic and for demonstration purposes only.
